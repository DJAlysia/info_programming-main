
# 메모리 ?
: 컴퓨터가 연산 및 프로그램 실행을 위해 기억하는 공간
- 프로그램이 실행되려면, 변수나 함수 등이 메모리공간에 할당되어야한다.
- Load(로드)        : 필요한 데이터들을 메모리에 가져오는 것
                     (Loading...)

# 메모리 구조
  1. 코드 영역      : 프로그램의 코드가 저장되는 영역

  2. 데이터 영역    : 전역변수, 정적변수가 저장되는 영역
                     - 할당 기간 : 프로그램 시작~종료

  3. 스택 영역      : 지역변수, 매개변수가 저장되는 영역
                     - 할당 기간 : 함수호출 ~ 함수종료

  4. 힙 영역        : 사용자가 직접 관리하는 영역
                     - 메모리 동적 할당
                      : 코드를 통해서 메모리를 할당 및 해제할 수 있다.
